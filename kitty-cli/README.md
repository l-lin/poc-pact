# kitty-cli

[![GoDoc](https://godoc.org/l-lin/kitty-cli?status.svg)](https://godoc.org/l-lin/kitty-cli)

> Simple CRUD client app that uses Pact as a consumer

## Getting started

```bash
# Build
make compile
```

## Usage

```bash
# Run binary
./bin/kitty-cli -h
# Or directly using go
go run .
```
